# 🚀 AI-Powered Tech News App

🌐 Live Demo: https://ai-news-backend-1nge.onrender.com/api/latest

📰 AI-powered technology news aggregation and summarization platform that collects, processes, and delivers real-time technology updates from global sources.

## Features

- Real-time technology news updates
- AI-generated news summaries
- Global technology coverage
- Category-wise filtering
- Mobile-friendly interface
- Automated news collection
An intelligent Android application that automatically gathers the latest technology news from global sources, processes the information using Artificial Intelligence, and delivers concise, easy-to-understand summaries to users in real time.

The application is designed to help technology enthusiasts, students, researchers, developers, and professionals stay updated with the rapidly evolving world of technology without spending hours reading lengthy articles from multiple websites.

---

## 📖 Overview

Technology advances at an unprecedented pace. Every day, new developments emerge in fields such as Artificial Intelligence, Robotics, Semiconductor Technology, Cybersecurity, Quantum Computing, Space Technology, Software Engineering, Consumer Electronics, and Scientific Research.

While information is widely available, finding reliable updates and reading numerous articles can be time-consuming.

This project addresses that challenge by creating an AI-powered news aggregation and summarization platform that automatically collects technology news from multiple trusted sources, generates concise summaries, categorizes content, and presents it through a clean and user-friendly mobile interface.

The goal is to provide users with the most important technological developments in a format that is quick to consume, informative, and accessible from anywhere.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Tech-News-App.git
```

### Navigate to Project Directory

```bash
cd AI-Tech-News-App
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure API Keys

Create a `.env` file and add:

```env
NEWS_API_KEY=your_api_key
OPENAI_API_KEY=your_api_key
```

### Run Application

```bash
python app.py
```
